# STM32 驱动

存放一些经过验证的驱动代码。主要是 STM32 外设和电机的驱动程序。

全部支持战队使用的[工程模板](https://github.com/Deadline039/STM32-Template)，添加到`BSP`层即可使用。

代码注释采用 doxygen 格式。后面会考虑将代码中的 doxygen 转换成 markdown，以方便查阅。

某些代码即使验证了也可能存在 bug，而且 README 中的 demo 不保证 100% 可用。欢迎 Issue 或 PR。



### `Motor`文件夹下：

| 模块      | 说明                                                       | 是否验证 |
| --------- | ---------------------------------------------------------- | -------- |
| CAN       | CAN 底层驱动以及回调链表，所有 CAN 通信的电机都需要依赖此模块 | 是       |
| AK-Motor  | CubeMars AK 系列电机                                        | 是       |
| DJI-Motor | 大疆 M3508/2006/GM6020                                      | 是       |
| VESC      | VESC 电调驱动                                       | 是       |
| step_motor | 定时器PWM驱动步进电机驱动代码 | 是 |
| MLDS      | 铭朗科技 伺服电机                                             | 否       |

### `Display`文件夹下：

| 模块      | 说明                                                        | 是否验证 |
| --------- | ----------------------------------------------------------- | -------- |
| OLED      | 支持0.91，0.96，1.3寸iic接口的oled基本驱动                  | 是       |
| OLED_RTOS | 支持0.91，0.96，1.3寸spi或iic接口的oled驱动，使用freertos。 | 是       |
| st77xx    | 1.8寸spi接口LCD屏幕驱动,支持st7735与st7789.                 | 是       |
